# 慧眼识瓷 - 宋煌官瓷数字化体验平台

> 器以载道 · AI赋能非遗技艺数字化传承

## 项目简介

"慧眼识瓷"是河南大学仝瓷记忆团队开发的宋代官瓷数字化体验平台。运用现代科技手段（3D建模、AR增强现实、AI识别等），将千年官瓷技艺进行数字化还原与呈现。

## 核心功能

- **沉浸式数字博物馆** - 72道工序展示、匠心工坊
- **官瓷通史** - 千年传承历史、交互式知识图谱
- **云端3D数字展馆** - Three.js 3D模型展示、360度沉浸式品鉴
- **AR实景体验** - AR扫一扫识别瓷器
- **展览预约** - 在线预约参观
- **在线商城** - 官瓷文创产品展示与购买
- **多语言支持** - 中文、English（即将支持日本語、한국어）

## 技术栈

- HTML5 + CSS3 + JavaScript
- Three.js (3D渲染)
- Font Awesome 6 (图标)
- 响应式设计

## 项目结构

```
├── 主代码.html          # 主页面
├── css/
│   └── style.css       # 样式文件
├── js/
│   ├── app.js          # 主要业务逻辑
│   ├── data.js         # 数据配置
│   ├── i18n.js         # 多语言支持
│   └── three-scene.js  # 3D场景渲染
└── images/             # 图片资源
```

## 使用说明

1. 直接在浏览器中打开 `主代码.html` 即可浏览
2. 推荐使用 Chrome、Edge 等现代浏览器
3. 3D展馆功能需要浏览器支持 WebGL
