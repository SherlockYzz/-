"慧眼识瓷"是河南大学仝瓷记忆团队开发的宋代官瓷数字化体验平台。 平台以"器以载道，AI赋能非遗技艺数字化传承"为核心理念，运用现代 科技手段（3D建模、AR增强现实、AI识别等），将千年官瓷技艺进行 数字化还原与呈现，让更多人能够了解和体验中国传统官瓷文化。本网页由于子倬，邢雅静同学制作，技术栈】 包括前端：HTML5 + CSS3 + JavaScript - 3D渲染：Three.js - 图标库：Font Awesome 6 - 样式：自定义CSS。【使用说明】 1. 直接在浏览器中打开"主代码.html"即可浏览 2. 推荐使用Chrome、Edge等现代浏览器 3. 3D展馆功能需要浏览器支持WebGL 4. AR功能需要后端服务支持和移动端摄像头权限
